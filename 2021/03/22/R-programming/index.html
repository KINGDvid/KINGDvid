R programming

● 벡터의 특징 및 강제변환의 뜻

I. 벡터는 R 객체 중 가장 기본이며 속성이 같은 집합으로 구성된다
II. 만약 속성이 서로 다르다면 상위 속성으로 강제변환 되며 이를 Coercion이라 부름
III. 논리형 ☞ 숫자형 ☞ 문자형

● 벡터 속성의 종류

  1. 숫자형(numeric): 정수, 실수
  2. 문자형(character): 문자열
  3. 논리형(logical): TRUE / FALSE
  4. 범주형(factor): 특정 유형 분류
    4.1. 명목형: 데이터 값 비교 불가
    4.2. 순서형: 데이터 값 순서 지정
